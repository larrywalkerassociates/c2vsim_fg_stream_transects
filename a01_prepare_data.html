Preparing Stream and Groundwater Data from C2VSimFG v1.5#

In this section, we prepare the model data that will be used to make the stream transects. The workflow used hereafter consists of the following steps:

  1. We load the Stream Specification File and the Nodal X-Y Coordinate File and generate a lines geodataframe of the streams represented in the model.

  1. Using the stream lines shapefile generated in the previous step, we calculate the distance of each stream node with respect to the origin of the stream line.

  1. We load the Stratigraphy File and join ground surface elevations and layer thicknesses to the stream nodes.

  2. We load the timeseries “Groundwater Head at All Nodes” file (C2VSimFG_GW_HeadAll.out) and join simulated groundwater heads to stream nodes.

  3. For each node, we extract the groundwater heads for the highest active layer. To do so, we iterate through the layers and select the highest one that presents a thickness greater than zero and is above the bottom of the layer.

  4. Select CASGEM wells and lithology logs within 500 m of Butte Creek.

  1. Select CASGEM wells and AEM lithology logs within 500 m of Butte Creek

  2. Download CASGEM water level observations for the selected wells.
